Prisma4Postgres

Cliente desktop PostgreSQL standalone construído com Electron. Gerencie bancos de dados com dashboard em tempo real, ERD automático e editor SQL Monaco.

  • Dashboard com métricas em tempo real
  • ERD gerado automaticamente
  • Editor SQL com Monaco
